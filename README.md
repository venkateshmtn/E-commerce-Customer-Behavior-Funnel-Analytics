## üìå Project Overview

This project analyzes **customer behavior on an e-commerce platform** to understand how users move through the purchase funnel, where they drop off, what drives conversions, and which customer segments contribute the most revenue.

The goal is to combine **business thinking + analytics** to generate **actionable revenue insights** that stakeholders can directly use to improve conversion rates, reduce churn, and increase customer lifetime value (CLV).

This project is designed as a **portfolio-ready, interview-focused analytics case study**.
